### 允许的突变类型

| 类型 | 说明 |
|------|------|
| **split** | 拆分任务为更小单元 |
| **insert** | 在已有任务之间插入新任务 |
| **skip** | 跳过不需要的任务 |
| **reorder** | 重排任务顺序 |
| **abandon** | 废弃任务（方向错误） |
| **strategic-replan** | 战略层面重规划（影响全表） |
| **rollback** | 回滚到前一次 strategic-replan 之前的战略 |
